# 🎓 MAKAUT CGPA Calculator

A simple and beginner-friendly **C++ console application** to calculate the **Average CGPA** and **Equivalent Percentage** for students of **MAKAUT (Maulana Abul Kalam Azad University of Technology), West Bengal**.

This project is designed for students who want a quick and accurate way to calculate their academic performance without performing manual calculations.

---

## ✨ Features

- 📚 Supports any number of semesters
- 📝 Accepts semester-wise CGPA input
- ✅ Input validation (CGPA must be between 0 and 10)
- 📊 Calculates Average CGPA
- 📈 Converts CGPA to Percentage
- 🔄 Allows multiple calculations in one execution
- 💻 Simple console-based interface
- 🚀 Beginner-friendly C++ code

---

## 🧮 Formula Used

### Average CGPA

```text
Average CGPA = (Sum of Semester CGPAs) / Number of Semesters
```

### Percentage

```text
Percentage = (Average CGPA - 0.5) × 10
```

> **Note:** This project uses the percentage conversion formula specified by the project requirements.

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/AnupamKumarPanditofficial/makaut_cgpa_calculator.git
```

### Navigate to the project

```bash
cd makaut_cgpa_calculator
```

### Compile

Using g++

```bash
g++ main.cpp -o cgpa_calculator
```

### Run

Windows

```bash
cgpa_calculator.exe
```

Linux/macOS

```bash
./cgpa_calculator
```
